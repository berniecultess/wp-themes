Installation

Installing the theme can be done two ways. You can upload the theme ZIP file using the built in WordPress theme upload, or copy the files through an FTP client.

 

1) Using WordPress Theme Upload

 

Be sure to extract the file "the-cause.zip" from the ThemeForest download before uploading. Using the ThemeForest ZIP file directly will result in a "Missing Style Sheet" error.

  1. Login to your WordPress admin.
  2. In the "Appearance > Themes" menu click the tab "Install Themes"
  3. At the top of the page click, "Upload", then click the file input to select a file.
  4. Select the zipped theme file, "the-cause.zip", and click "Install Now"
  5. After installation you will receive a success message confirming your new install.
  6. Click the link "Activate"

2) Uploading by FTP

  1. Login to your FTP server and navigate to your WordPress themes directory. 
    Normally this would be "wp-content/themes"
  2. Extract the files from the zipped theme.
  3. Copy the folder "the-cause" to your themes directory.
  4. After the files finish uploading, login to your WordPress admin.
  5. In the "Appearance" menu click "Themes"
  6. Click "Activate" for the theme "The Cause"

Theme Options

If you want to change the general Options of the theme open the "The Cause Options" panel (located in your admin sidebar) and click it, you will notice several tabs with options, which we will discuss now one by one.

1) General Options

Dashboard -> The Cause Options -> The Cause Options

In this area you can change all logos, favicon, default avatars or default thumbs.

If you want to upload your own logo, just click on upload and choose an image from your computer or media library. There are no restrictions, but please make sure that your image isn't wider than 979px.

NOTE: please make sure to use full size of image (following screenshot)

 

If you decide to have custom login page (with your own logo and background), then you must choose 'yes' as it is described on the first screenshot.

NOTE: please use this option ONLY if your server has cURL extension enabled or allow_url_fopen enabled.

You can upload logo for your login page (same as above) or to choose custom background color/image (see the first screenshot)

 


You can upload your set of image.
For your own custom branding the most important images are favicon, menu icon and default avatar (if you decide to use comments)

Final set of general settings. The most important part is footer copyright line which will be presented in bottom left corner of your footer.

If you decide to use landing page (page with 'Coming soon' text and countdown, then you should populate 'Countdown - Final Date' field. 

Format:

$year, $month, $day, $hours, $minutes, $second

(i.e. if you want to countdown till 6th of November 2012, 1:30PM, then you will populate appropriate field with following data: 2012, 11, 6, 13, 30, 0)

2) Appearance Settings

Dashboard -> The Cause Options -> Appearance settings

In this section you can easily change backround image and background color of header (area above main navigation) or landing page (if you decide to use one). Dimesions are optional, but we recommend you to use seamless patterns if your images are smaller than 2000px in width.

You can also change main background image or choose to add shadow gradient on top of it. You can use seamless patterns here, because main image is repeatable.
NOTE: you can set different background image for each page of your site!

The most important part of this section is for sure changing of all inner colors (navigation, buttons, sidebar headings). You can choose one of 3 predefined color sets or to make your own custom set.

Fonts are located in the-cause/styles/font (for regular) and the-cause/styles/font2  for cursive font.

If you want to add new font from Google collection, just follow these instruction:

- create new css file and put it in appropriate folder (font or font2)
- name it after chosen font (i.e. lato.css; filename will be used in this example)
- navigate to http://www.google.com/webfonts and find Lato
- click on quick use
- choose the styles you want (we have chosen Normal 400 and Bold 700)
- in the box 3 choose @import tab and copy its content into lato.css
- in the box 4 you will find appropriate line to be used for all objects generated with custom font (please check styles/fonts/default.css for default file content)
- font will appear in dropdown
 
 
IMPORTANT NOTE: Be aware that some fonts appear larger than others at the same point size. So, if you make your own font file, please check size of fonts in the main style file (style.css)
 

You can also change feel and look of important sliders (on home page and media pages).
 
Home page slider will be used on page with SlideJS plugin (Template Name: Home). It will use slides (check Custom post types). You can:
  • choose order of slides (menu order or random)
  • chose type of effect (slide or fade)
  • choose speed of animation and pause of slide
  • decide do you want to use easing effect (and type of effect)
Media sliders are used on video pages and gallery pages. You can:
 
  • chose type of effect (slide or fade)
  • choose speed of animation and pause of slide
  • decide do you want to use easing effect (and type of effect)

NOTE: list of types of easing effects you can find on this URL: http://gsgd.co.uk/sandbox/jquery/easing/

 

3) Home/Archive

In this area you can change settings for several archive pages.

A) Home Page Settings

  1. You can set homepage slogan or leave it blank if you don't want to use it
  2. You can set custom text for donate button (if you already chose donations page. See Pages and Categories)
  3. Choose homepage areas and order of them. You can decide between: Latest posts, Videos and Page content.
  4. If you decide to use Latest posts on homepage you can set number of them
  5. If you decide to use Videos on homepage you can set number of them

B) Blog Settings

  1. Blog page layout: wide or 3 columns (this option will be overriden if you choose one of specific templates for category page). This setting will be used for all other archive pages.
  2. Number of items - wide style: you can choose values between 3 and 12. If you have more articles than chosen number, pagination will be automatically added.
  3. Number of items - columns style: you can choose values between 3 and 15. If you have more articles than chosen number, pagination will be automatically added.
  4. Exclude categories: add ID of category. If you want to exclude several categories, separate IDs with commas (i.e. 3,4,12). This will be used ONLY on blog pages selected without specific category selected.

NOTE: archive pages will use number of posts set in WordPress options (Dashboard -> Settings -> Reading -> Blog pages show at most), so if you want to use columns layout, we strongly recommend number that can be divided by 3 (3, 6, 9, 12, ...). See screenshot:

C) Media Settings

  1. Show featured slider: if you enable this option, each gallery/media/video page will contain slider with featured items on top.
  2. Number of photos in featured slider: you can choose value between 3 and 9. If you have more featured photos than chosen value, some of featured photos will be excluded
  3. Number of thumbnails: you can choose values between 3 and 15. If you have more photos than chosen number, pagination will be automatically added.
  4. Number of videos - wide style: you can choose values between 3 and 10. If you have more videos than chosen number, pagination will be automatically added.
  5. Number of videos - columns style: you can choose values between 4 and 16. If you have more videos than chosen number, pagination will be automatically added.

4) Pages and Categories

In this area you can set important pages used across your site.

The most important part of this area is setting of donations page. You can decide whether to use page from your own site or external link. Page from site will have priority!!!

5) About Us

In this area you can set candidate biography or movement info which will be used in sidebar if you choose one of premade custom widgets. You can also set your contact details, social network profiles, location of office, etc...

NOTE: email set in this area will be used for contact form, newsletter sign up and event attendence application.

IMPORTANT NOTE

Theme will use predefined values for all theme settings if you don't provide yours.

Custom Post Types

The Cause WordPress Theme comes with 6 custom post types. Files with their definitions are located in the-cause/includes/posttype, If you wish to add your custom post type, just put definition file in this folder and theme will automatically load it.

All custom post types menus are located in admin dashboard.

1) Events

This custom post type is used for campaign events.

Every event has following attributes:

  1. Title
  2. Content
  3. Featured Image
  4. Location (i.e. Chicago)
  5. Venue (i.e. Hotel Candidate)
  6. Start date
  7. End date
  8. Time of event
  9. Extra photos (it will be used for event gallery)

If you decide to show event location on Google Map, at least one of map attributes (latitude/longitude) should be different from zero. Google Map attributes are:

  1. Latitude
  2. Longitude
  3. Zoom

We recommend this site for descovering desirable latitudes and longitudes: http://universimmedia.pagesperso-orange.fr/geo/loc.htm

2) FAQ

This custom type is used for FAQ pages.

Format:

  1. QUESTION: post title
  2. ANSWER: post content

Questions are ordered by menu order ascending. Just put proper number in order field (located in right hand sidebar) and that is it.

3) Photos

This custom post type is used for photo items.

Every photo has following attributes:

  1. Title
  2. Featured Image: (minimum dimensions: 920 x 350 in pixels if you want to show photo in featured image slider)
  3. Featured: YES or NO. If you choose 'YES', photo will be shown on home page or sidebar (if appropriate widget is activated) or in gallery page slider.

4) Slides

This custom post type is used for home page slider.

Every slide has following attributes:

  1. Title
  2. Content (optional)
  3. Featured image (minimum dimensions: 940 x 310 in pixels)
  4. URL (manually added link or chosen from list of pages)
  5. Menu order (it will be used if theme options requires order by menu order in home page slider)

Don't put images in slider content! Please use featured image for slider image.

If you populate manual URL field and select a page from dropdown, URL will be manually added link.

5) Testimonials

This custom post type is used for quotes of your clients and customers.

Every testimonial has following attributes:

  1. Title
  2. Content
  3. Name (of client/customer)

6) Videos

This custom post type is used for video items.

Every video has following attributes:

  1. Title
  2. Content
  3. Featured Image: (minimum dimensions: 444 x 237 in pixels)
  4. Featured: YES or NO. If you choose 'YES', video will be shown on home page.
  5. URL: URL of your Youtube/Vimeo movie

YouTube format: http://www.youtube.com/watch?v=ID
Vimeo format: http://vimeo.com/ID

If you want to use a default image of Youtube/Vimeo video, leave featured image blank and theme will use an image from appropriate video service.

Shortcodes

Here is a list of shortcodes that can be used in our theme.

DIVIDERS

Usage: [{dividerName}]
{dividerName} - hor, hor2, horDashed, horDouble, horShadow, starHor

[hor] - outputs solid  horizontal line
[hor2] - outputs solid horizontal line - version 2
[horDashed] - outputs dashed horizontal line
[horDouble] - outputs double horizontal line
[starHor] - outputs divider with stars
 

CLEAR

Usage: [clear height={height}]
{height} - int type

[clear] - clears formating
[clear height=100] - height of this element will be 100px.

 

BACK TO TOP BUTTON

Usage: [top title={title} float={float} round={round}]
{title} - string type
{float} - left:right:none
{round} = normal:round:roundX

It will make a tiny button which scrolls page to the top.

Example:
[top title="Top of page" float="right" round="round"]
will output right floated rounded tiny button.

 

BUTTONS

Usage: [button size={size} url={url} float={float} round={round}]{content}[/button]
{content} - string type
{size} - normal:big:tiny
{url} - URL
{float} - left:right:none
{round} - normal:round:roundX

 
This shortcode generates a button.
 
Example:
[button size="big" url="home.htm" float="right" round="roundX"]My Link[/button]
will output right floated extra rounded big button entitled 'My Link'. It will link page with page home.htm

Color of button is controlled by theme options (button style).

 

 

INFO BOXES

 

Usage: [box color={color}]{content}[/box]
{content} - string type
{color} - green|red|yellow|blue

 
This shortcode generates an info box.
 
Example:
[box color="yellow"]Warning Message[/box]
will output yellow info box with 'Warning Message' in it.
 

LINK TO CONTACT PAGE

If you set contact page through theme options (The Cause > Pages and Categories), then you can use following shortcode which will generate button with custom title linked to contact page.

 

Usage: [contact size={size} float={float} round={round}]{content}[/contact]
{content} - string type
{size} - normal:big:tiny
{float} - left:right:none
{round} - normal:round:roundX

Example:
[contact size="tiny" float="right" round="roundX"]Link to Contact page[/contact]
will output right floated extra rounded tiny button entitled 'Link to Contact page'
